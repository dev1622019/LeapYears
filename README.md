# Prerequisites
Java should be installed on the system


# Compilation

1. Clone the project
2. Open the terminal
3. Navigate to the folder of the project 
4. Run the following command:
#### Linux/OSX
```./gradlew shadowJar```
#### Windows
```gradle.bat shadowJar```

# Execution
Run the following command:  
```java -jar build/libs/162-1.0-SNAPSHOT-all.jar```
